## Project Description

Backend application built with **Node.js** and **Express.js**, using **WebSocket** for real-time communication.
The backend handles media data, manages user connections, and communicates with the frontend for media playback and recording.

## Technologies

* Node.js
* Express.js
* WebSocket
* JavaScript (ES6+)

## Installation & Run

```bash
npm install
nodemon webs.js
```


## Project Structure

```
/backend
  webs.js
   package.json
   README.md
```

## Features

* REST API endpoints for media handling
* WebSocket server for real-time communication
* Media file management
* Handles multiple simultaneous connections

Demonstrate skills in backend development with Node.js, Express, and WebSocket for a fullstack engineering task.
